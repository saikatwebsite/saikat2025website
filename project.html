Self~Made Portable Computer using Raspberry Pi 4

The project is a self-designed portable computer, It uses Linux based operating system "Kali Linux", There is a 6-inch touchscreen display for obvious reasons, it also has touch screen support. The power management is done using a powerbank with 10000 mah battery back-up to ensure portability. The project is made using Raspberry Pi 4. There is a wireless keyboard with touch-pad for input purposes.

Routine || Schedule Generator for Colleges , School

The project is a routine or schedule generator for colleges , schools. The purposes of the project is to generate routines based on the subject credits, students enrolled, teachers available and sitting capacity of the rooms, which are vital necessities for any college routine or school routine. The project can be tailored for other applications as well so as to generate routine or schedules as needed. The project is developed in C++
